# Formula 1 2025 Abu Dhabi Grand Prix analysis using SQL

## Data Description
The dataset consists of three CSV files from Kaggle, slightly modified in Excel (only some column names were adjusted):  
1) **lap_times.csv** - lap-by-lap performance data  
2) **qualifying_results.csv** - qualifying session results (Q1, Q2, Q3)  
3) **race_results.csv** - final race standings with detailed race information
